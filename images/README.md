# Images nécessaires pour le portfolio

Pour que le portfolio fonctionne parfaitement, vous devez ajouter les images suivantes dans le dossier `images/` :

## Images obligatoires :

### Photo de profil
- **profile.jpg** (300x300px recommandé)
  - Photo professionnelle du webdesigner
  - Format carré, haute qualité
  - Arrière-plan neutre de préférence

### Images de projets
1. **project1.jpg** - Restaurant La Belle Époque (16:10 aspect ratio)
2. **project2.jpg** - Application TaskManager Pro (16:10 aspect ratio)
3. **project3.jpg** - Boutique FashionStyle (16:10 aspect ratio)
4. **project4.jpg** - Cabinet d'Architecte (16:10 aspect ratio)
5. **project5.jpg** - Branding TechStart (16:10 aspect ratio)
6. **project6.jpg** - Application WeatherNow (16:10 aspect ratio)

## Dimensions recommandées :
- **Profile photo** : 300x300px (carré)
- **Project images** : 800x500px (ratio 16:10)
- **Format** : JPG ou PNG
- **Qualité** : Haute résolution pour les écrans Retina

## Sources d'images gratuites :
- [Unsplash](https://unsplash.com/) - Photos libres de droits
- [Pexels](https://pexels.com/) - Photos et images gratuites
- [Pixabay](https://pixabay.com/) - Images libres de droits
- [Freepik](https://freepik.com/) - Ressources graphiques

## Alternative : Images placeholder
Si vous n'avez pas d'images pour le moment, vous pouvez utiliser des services de placeholder :
- `https://via.placeholder.com/300x300/6c5ce7/ffffff?text=Profile`
- `https://via.placeholder.com/800x500/6c5ce7/ffffff?text=Project+1`

## Optimisation :
- Compressez vos images avec des outils comme TinyPNG
- Utilisez le format WebP quand possible pour de meilleures performances
- Assurez-vous que les images ne dépassent pas 500KB chacune
